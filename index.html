Code Weekend

Building your first webapp

Code Weekend is scheduled for September 6th and 7th, 2014 and will comprise four workshops to help participants build their very own webapp. Participants are expected to have very minimal coding exposure as we will be carefully explaining everything we do as we go. This should be extremely helpful to first time hackers especially with PennApps scheduled for the next weekend. If you haven't already, sign up here. There is also a high likelihood of there being some food around.

<What we'll cover />

Over the course of the weekend we will be building an app that does ______________. We will start by setting up an environment in which to work, followed by an Intro to NodeJS and Javascript, which we will be using to build this app. This will be followed by sessions in which we will add functionality to this basic app we built by using third party-APIs and by allowing data to persist on a Database (MongoDB in this case).

<Who we are />

This workshop series has been organized by PennCC (formerly known as The Dining Philosophers) in association with PennApps and with help from a lot of really smart individuals. The workshops will be led by students, and there will be student mentors around to help out participants with their code and setup.
